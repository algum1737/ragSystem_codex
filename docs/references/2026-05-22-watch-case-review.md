# 2026-05-22 Watch Case Review

## Summary

source drift regression report의 watch case 7건을 검토했다. 결론은 `eval/test_cases.json`을 즉시 보정하지 않고, watch 상태로 유지하는 것이다.

이유:

- 7건 모두 `answer_accuracy=1.0`, `faithfulness=1.0`이다.
- 현재 문제는 답변 실패가 아니라 `relevant_sources`와 확장 corpus의 동등 근거 사이의 정렬 문제다.
- 최신 retrieved source를 그대로 `relevant_sources`에 추가하면 source metric은 올라가지만, 평가셋이 현재 검색 결과에 과적합될 수 있다.

## Decision

- `critical` case는 계속 guard 실패 조건으로 둔다.
- `watch` case는 CI 실패 조건으로 승격하지 않는다.
- watch case는 평가셋 보정 전 직접 근거성 검토가 필요한 후보로 유지한다.
- 이번 작업에서는 `eval/test_cases.json`을 변경하지 않는다.
