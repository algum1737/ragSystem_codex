1. 유료서비스 2개
2. 운영정책 2개
3. 위치기반서비스 1개
4. negative/partial-answer 1개

확장 후 full eval을 재실행해 다음을 비교한다.

- 기존 10개 케이스 점수 유지 여부

- `rag_normalized_source_precision@k_mean`
- `rag_chunk_precision@k_mean`
