# ragSystem 서버 기동 가이드

## 구성 요소

| 컴포넌트 | 포트 | 역할 |
|----------|------|------|
| Ollama | 11434 | LLM 실행 엔진 (gemma3:12b) |
| FastAPI | 8000 | Python API 서버 |
| Streamlit | 8501 | Web UI |

---

## 1. Ollama (LLM 서버)

Homebrew 서비스로 등록되어 있어 **Mac 시작 시 자동으로 켜집니다.**

```bash
# 상태 확인
brew services list | grep ollama

# 수동 시작
brew services start ollama

# 중지
brew services stop ollama

# 재시작
brew services start ollama

# 설치된 모델 확인
ollama list
```

> 정상 실행 확인: `curl http://localhost:11434/api/tags`

## 2. FastAPI 서버

```bash
cd /Users/hun/workspace/ragSystem
source .venv/bin/activate
uvicorn api.main:app --port 8000
```

백그라운드 실행:
```bash
source .venv/bin/activate && uvicorn api.main:app --port 8000 &
```

> 정상 실행 확인: `curl http://localhost:8000/health`

---

## 3. Streamlit Web UI

```bash
cd /Users/hun/workspace/ragSystem
source .venv/bin/activate
streamlit run app.py
```

백그라운드 실행:
```bash
source .venv/bin/activate && streamlit run app.py --server.headless true &
```

> 브라우저: http://localhost:8501

## 전체 순서 (한 번에)

```bash
cd /Users/hun/workspace/ragSystem
# Ollama는 자동 실행 중 (확인만)
curl -s http://localhost:11434/api/tags > /dev/null && echo "Ollama OK"

source .venv/bin/activate

# FastAPI
uvicorn api.main:app --port 8000 &

# Streamlit (FastAPI 시작 후 3초 대기)
sleep 3 && streamlit run app.py --server.headless true &
```

---

## 중지

```bash
# FastAPI / Streamlit 중지
kill $(lsof -ti:8000) 2>/dev/null
kill $(lsof -ti:8501) 2>/dev/null

# Ollama 중지 (필요 시)
brew services stop ollama
```

## 탭별 기능

| 탭 | 기능 |
|----|------|
| 인제스천 | PDF / DOCX / DOC / HWP 문서를 벡터 DB에 저장 |
| 쿼리 | 저장된 문서 기반 RAG 질의응답 |
| 시스템 | 서버 상태 확인, **Ollama 모델 변경** |
| PPT 생성 | RFP 업로드 → AI 제안서 초안 자동 생성 → PPTX 다운로드 |

## 모델 변경

시스템 탭에서 드롭다운으로 선택하거나 API로 직접 변경:

```bash
curl -X PUT http://localhost:8000/model \
  -H "Content-Type: application/json" \
  -d '{"model": "gemma3:12b"}'
```

> 서버 재시작 시 `gemma3:12b`(기본값)로 초기화됩니다.
